# Research

This directory is reserved for future source research, technology evaluations, editorial experiments, and implementation notes that are not yet canonical decisions.

Rules:
- Move durable conclusions into `docs/specification.md`, `docs/delivery/execution.md`, or `docs/delivery/decision-log.md`.
- Keep this directory for working notes, comparisons, and short-lived investigations.
- Do not keep parallel legacy PRDs here once canonical docs cover the same ground.
- Do not store secrets or exported production data here.
